## ⚙️ How to Use StartAllBack

After installation, you will notice the Start menu and taskbar have changed. To customize further, follow these steps.

### Open the settings

Right-click the Start button (or press `Win + X`) and select **Properties**. This opens the StartAllBack Control Panel.

### Change the Start menu style

In the **Start Menu** tab, you can switch between Windows 10, Windows 7, or Custom. Each style has its own options. For example, under Windows 7, you can turn off the “All Programs” menu or change the number of recent programs shown.

### Adjust the taskbar

Go to the **Taskbar** tab. Here you can:

- Move the taskbar to any side of the screen.
- Choose small or large icons.
- Combine taskbar buttons always, only when full, or never.
- Show seconds on the clock.
- Disable the “News and interests” widget.

### Modify File Explorer

Switch to the **File Explorer** tab. You can:

- Use the classic ribbon (like Office 2010) or a simpler command bar.
- Remove the “Gallery” and “Home” sections from the navigation pane.
- Turn on the old navigation pane style.

### Restore old context menus

In the **Advanced** tab, check the box that says **“Use classic full context menu”**. This removes the extra “Show more options” step in Windows 11.

### Apply changes

Every change takes effect immediately. You do not need to restart your computer or sign out.

## 🔧 Troubleshooting

StartAllBack works reliably, but if you run into issues, try these steps.

### Start menu does not open

- Restart Windows Explorer. Press `Ctrl + Shift + Esc` to open Task Manager, find **Windows Explorer**, right-click it, and choose **Restart**.
- If that does not help, open StartAllBack settings and select a different Start menu style. Then switch back.

### Taskbar disappears

- Right-click an empty area of the taskbar and uncheck **Lock the taskbar**. Then drag the edge to resize it.
- In StartAllBack settings, go to the **Taskbar** tab and change the **Taskbar location** to **Bottom**.

### Uninstall StartAllBack

You can remove it anytime. Go to **Settings > Apps > Installed apps** (Windows 11) or **Control Panel > Programs and Features** (Windows 10). Find **StartAllBack** in the list, click **Uninstall**, and follow the prompts. Your system goes back to the default Windows look.

### The installer says I need more permissions

Make sure you are logged in with an administrator account. If you are, right-click the installer and choose **Run as administrator**.

### Something else

Visit the support page on GitHub. You can open an issue in the repository and ask for help. Include your Windows version and a short description of the problem.
